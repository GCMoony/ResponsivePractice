Adding Numbers

You can add defined variables together in Javascript.

Below is an example of 2 defined variables being added and the value being stored in a third variable.

let a = 1;
let b = 2;
let c = a + b;
console.log(c); // Expected output: "3"

What did I just do?

  • I instatiated two variables with values
  • I instantiated a third variable using with the sum value of the other two variables

Exponents

You can retrieve a value of a defined variable to the power of another defined variable

Below is an example of 2 defined variables where one value is the base, and the second value is the power, and the value being stored in a third variable.

let a = 8;
let b = 2;
let c = 8 ** 2;
console.log(c); // Expected output: "64"

What did I just do?

  • I instatiated two variables with values
  • I instantiated a third variable using the the value of one variable to the power of the other variable
